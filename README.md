# GeoBot
Личный бот для создания, получения, хранения, обработки заявок для геодезии
Возможности:
- Сощлание заявки с 4-мя видами состояний
- Созданная заявка имеет в себе достаточно информаци для выполнения работы и связи с отправителем
- Указывает видео отправления и время её подтверждения
- Данные о заявках и пользователях хранятся в базе данных
